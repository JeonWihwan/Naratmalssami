## Project setup


## 프로젝트 내용
* 저희 **나랏말싸미** 는 한글로 소통하는 공간입니다.
* 한글보다 외국어, 외래어가 익숙한 사람들에게 한글의 아름다움을 느낄 수 있도록 한국 전통을 테마로 한 웹 페이지입니다.
* 회원관리, 게시판 CRUD, 로그인/아웃, push알람, image 업로드, 챗봇, 메일전송 등의 기능을 구현하였습니다.
